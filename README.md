# Whipbrick Website

Site corporate officiel de Whipbrick - Consultant Data Analytics.

## Structure

- **index.html** : Page d'accueil
- **services.html** : Services Data Analytics
- **people-analytics.html** : Services People Analytics
- **expertise.html** : Expertise & Bio
- **contact.html** : Contact

## Stack

- HTML5
- CSS3 (variables CSS, Grid, Flexbox)
- JavaScript vanilla
